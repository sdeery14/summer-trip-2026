# Summer Trip 2026 — Greater Yellowstone Loop

A ~7-day car camping and hiking trip for two, late August 2026. Fly from Boston, rent a car, loop through Yellowstone, Beartooth Highway, and Grand Tetons.

**Route:** BZN (Bozeman) → Yellowstone NP → Beartooth Highway → Red Lodge → back into Yellowstone → Grand Teton NP → JAC (Jackson Hole)

---

## Quick Links — Action Items

These are the things you need to do, roughly in order:

### Right Now (March 2026)
- [ ] [Yellowstone backcountry permits](https://www.nps.gov/yell/planyourvisit/backcountryhiking.htm) — check if lottery is open
- [ ] [Grand Teton backcountry permits](https://www.nps.gov/grte/planyourvisit/back.htm) — check advance reservations
- [ ] [Reserve campsites on recreation.gov](https://www.recreation.gov/) — Canyon, Mammoth, Signal Mountain, Colter Bay
- [ ] [Buy America the Beautiful Pass ($80)](https://store.usgs.gov/pass/index.html) — covers both parks

### April–May
- [ ] [Book flights on Google Flights](https://www.google.com/flights) — BOS → BZN + JAC → BOS (open-jaw)
- [ ] Book rental car at BZN — midsize SUV (RAV4/Outback class), one-way to JAC

### 1–2 Weeks Before
- [ ] Call Yostmark in Driggs for crash pad rental — (208) 354-2828
- [ ] Book rafting trip if wanted — [Glacier Raft Co](https://www.glacierraftco.com/) or [Snake River options](features/biking-and-rafting/output.md)
- [ ] Check [Beartooth Highway status](https://www.511mt.net/)
- [ ] Check [Yellowstone road status](https://www.nps.gov/yell/planyourvisit/roads.htm)
- [ ] Check [AirNow for smoke](https://www.airnow.gov/)

### Day of Arrival (Bozeman)
- [ ] Buy bear spray x2 — REI Bozeman or Schnees
- [ ] Buy fuel canisters for camp stove
- [ ] Walmart grocery run — N 7th Ave, Bozeman

## The Itinerary at a Glance

| Day | Where | Sleep | Focus |
|-----|-------|-------|-------|
| **0** | Fly in → Bozeman → Gardiner | Gardiner area | Travel, bear spray, groceries, Boiling River soak |
| **1** | Yellowstone North | Slough Creek / Tower Fall | Dawn Lamar Valley wildlife, day hike |
| **2** | Beartooth Highway | Red Lodge or Beartooth Lake | Drive the highway, alpine hike, Red Lodge brewery |
| **3** | Yellowstone Core | Canyon / Bridge Bay | Grand Canyon of Yellowstone, geothermal, summit hike |
| **4** | Yellowstone → Tetons | Jenny Lake / Gros Ventre | Morning hike, drive south, settle into Tetons |
| **5** | Grand Tetons — Big Day | Same camp | Paintbrush-Cascade loop or Table Mountain |
| **6** | Grand Tetons — Flex | Same camp or Jackson | Bouldering, second hike, rafting, or chill day |
| **7** | Departure | Fly home from JAC | Schwabacher sunrise, fly out |
